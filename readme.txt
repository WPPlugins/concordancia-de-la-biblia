== Description ==Utilice el cuadro de búsqueda para insertar cualquier palabra, puede buscar todos las versiones de la biblia disponibles, o elegir una individual.
== Installation ==Puedes colocar la concordancia de la biblia sin necesidad de instalar este plugin, con solo copiar este codigo en tu widget como Texto arbitrario o HTML:`<iframe frameborder="0" scrolling="no" width="100%" height="200px" border="0" src="https://www.bibliatodo.com/online/concordancia"></iframe>`o puedes instalar el el plugin y seguir estas indicaciones
1. Sube el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que se ve correctamente!
   - Widget: Se puede añadir un widget a la barra lateral de la Concordancia de la Biblia, valla a apariencia - widget "Concordancia de la Biblia"
   - Replace tags: Se puede utilizar el siguiente código corto en cualquier entrada o pagina: [cdlb_concordancia_biblia] .
   - Template: en cualquier lugar de su plantilla inserte este codigo php `<?php echo cdlb_concordancia_funcion_de_la_biblia(0); ?>` .
